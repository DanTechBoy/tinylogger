> A tiny logger tool for Node.js.

## Install
`npm install`

## Usage
```javascript
const logger = require("dtblogger");

logger.info("Hello!");
logger.error("Something went wrong!");
logger.debug("Debugging!");
logger.fatal("Fatal error occurred!")
```
